# Architecture Diagrams

Mermaid diagrams for the Client Management API. Each file contains a single diagram.

| File | Description |
|------|-------------|
| [architecture.mmd](architecture.mmd) | Layer architecture (Presentation → Business → Data Access → Integration) |
| [components.mmd](components.mmd) | Class diagram showing all main components and their relationships |
| [database-schema.mmd](database-schema.mmd) | Entity-Relationship diagram of the clients table |
| [flow-create-client.mmd](flow-create-client.mmd) | Flowchart for the client creation process |
| [flow-update-client.mmd](flow-update-client.mmd) | Flowchart for the client update process |
| [request-sequence.mmd](request-sequence.mmd) | Sequence diagram showing Create and Soft Delete interactions |

## How to view

- **Mermaid Live Editor**: Paste the contents at [mermaid.live](https://mermaid.live)
- **GitHub / GitLab**: Renders `.mmd` files natively in Markdown when referenced as code blocks
- **VS Code**: Install the "Mermaid Preview" extension
